# Maintainers

## Primary maintainer

- GitHub: [@BootcampOnbaco](https://github.com/BootcampOnbaco)
- Role: primary maintainer
- Responsibilities:
  - Scope and roadmap.
  - Issue and pull-request triage.
  - Security response.
  - Release preparation.
  - Test and CI health.
  - Rule-quality and false-positive review.

## Adding maintainers

Maintainer access may be granted after sustained, high-quality contributions,
responsible security behavior, and demonstrated understanding of the project's
local-first publication-safety model.
